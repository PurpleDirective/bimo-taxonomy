This taxonomy is built from primary FDA sources where directly accessible, with secondary analysis from legal and regulatory firms (Goodwin, FDLI, Cooley) used to confirm and contextualize. Every claim links to a primary source. Known limitations are disclosed in the Methodology section of the taxonomy itself.
